# 表盘选择(clockpicker)
该组件是进行时分选择，点击输入框时会弹出一个表盘，选择的时间默认会以“HH:mm”的格式返回，并显示在输入框中。
##如何使用
1、定义样式为“u-clockpicker”的div元素。
	
	<div class='u-clockpicker'>
        
    </div>
2、在第一步创建的div元素中添加样式为“u-input”的输入框。用户选择的时间将会存放在这里。

	<div class='u-clockpicker'>
        <input class="u-input" type="text">
    </div>
